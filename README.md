# High School AI Learning - Java Project

## Introduction

This is an **Artificial Intelligence learning project** for high school students, implementing various AI algorithms and concepts using **Java** programming language.

## Learning Goals

- Understand AI fundamentals
- Master Java programming skills
- Implement classic AI algorithms hands-on
- Develop computational thinking

## Project Structure

```
ai-java-learning/
├── README.md          # Project documentation
├── .gitignore         # Git ignore file
├── LICENSE            # MIT License
├── src/               # Source code
│   └── com/josh/ai/
│       ├── HelloAI.java           # First AI program
│       ├── Perceptron.java        # Neural network
│       └── NumberGuessingAI.java  # AI game
```

## Quick Start

1. Clone the repository
```bash
git clone https://github.com/joshzhao0218/ai-java-learning.git
cd ai-java-learning
```

2. Compile and run examples
```bash
cd src/com/josh/ai
javac *.java
java HelloAI
java Perceptron
java NumberGuessingAI
```

## Example Codes
